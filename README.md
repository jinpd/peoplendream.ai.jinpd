# Foreigner Civil Service Chatbot Landing Page

This is the landing page for the **Foreigner Civil Service Chatbot**, a service designed to help foreigners in Korea navigate complex administrative tasks using an AI-powered chatbot (NotebookLM).

## Features
- **One-Page Design**: Clean, modern, and trustworthy layout.
- **Multilingual Support**: English and Korean copy for better accessibility.
- **Responsive**: Fully optimized for desktop and mobile devices.
- **Direct Integration**: Links directly to the NotebookLM chatbot interface.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to Deploy
1. Upload these files to a web server or a static hosting service (e.g., GitHub Pages, Netlify, Vercel).
2. Ensure the `assets` folder is included so images load correctly.
